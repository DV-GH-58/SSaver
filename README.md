# SSaver


Screensaver with background picture, 
time and date and RSS feed

Single executable file for Windows
Can be used as a screensaver .src or just run in fullscreen mode

## Issues

Does not work as screensaver right now, 
but works perfect in fullscreen mode

## Future Usage
Run SSaver.exe with parameters:
`/s` - run as screensaver (not working right now)
`/f` - run in fullscreen mode
`/c` - open configuration dialog
